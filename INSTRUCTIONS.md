# AgenticNow 内容系统指南

## 账号定位

@AgenticNow 追踪 Agentic Web 的建设者、系统和转折信号。核心立场：数据、身份、资产应该由个人掌控，而不是被科技巨头和中心化组织垄断。

核心论点：
- Crypto = AI Agent 的原生货币
- DAO = Agent 原生组织形态
- 互联网演化路径：信息 → 价值 → 意图（Agentic Web）

## 出手条件

不是每天都发。只在三种情况下发推：

1. 趋势拐点：Telegram 日报连续多天出现同一信号，形成 pattern
2. 有真实判断：读完某篇文章后有自己的观点，不是转述
3. Vibe coding 洞察：非程序员用 AI 构建产品的反直觉经验

## 英文风格

- 判断先行，不做资讯搬运
- 禁用词：revolutionary, game-changing, groundbreaking, exciting
- 开头10个词必须抓人
- 链接放 self-reply，不放主推文
- 用 open question 收尾，引导回复
- Thread 3-5条为宜，不超过7条

## 内容格式模板

### 趋势拐点型
This week's strongest signal from [N] sources: [one-line summary]
→ 2-3句展开
→ 连接到 Agentic Web 大叙事
→ Open question

### 判断先行型
[Bold take]
→ Here's why（3 bullets max）
→ The bigger picture
→ Agree? Disagree?
